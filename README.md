# Temple's "Entropy" claim, broken down by first principles

An interactive, source-backed teardown of Deepinder Goyal's claim that his wearable "Temple" measures the real-time cost of being alive via a metric called Entropy.

It separates what holds up (the body really is an energy-dissipating system), what is overstated (the 1908 rate-of-living theory), and what is unproven (that you should chase a lower resting number).

Every scientific claim links to peer-reviewed research: the Baltimore Longitudinal Study of Aging, Speakman's metabolism and lifespan review (Journal of Experimental Biology), the naked mole-rat study (eLife), the JAMA Network Open VO2 max cohort, and more.

Open `index.html` in any browser. It is a single self-contained file.

This is independent analysis for understanding the claims, not medical advice.
